# Deep Engine

A deep, bassy engine sound mod for Carrier Command 2.

# Development

1. Install ffmpeg w/libvorbis
2. Checkout source to %appdata%\Carrier Command 2\mods
3. Run build.bat
4. Launch Carrier Command 2 and enable the mod
